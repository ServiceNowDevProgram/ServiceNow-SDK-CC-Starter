## ATF Categories and APIs

Fluent includes structured ATF APIs for the following test step categories:

- `atf.appnav` - Application Navigator
- `atf.form` - Form
- `atf.form_sp` - Forms in Service Portal
- `atf.rest` - REST
- `atf.server` - Server-side logic
- `atf.catalog` - Service Catalog
- `atf.catalog_sp` - Catalog in Service Portal
- `atf.email` - Email
- `atf.reporting` - Reporting
- `atf.responsiveDashboard` - Responsive Dashboards
